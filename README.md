# android_device_asus_X01BD
For building TWRP for asuszenfone max pro m2 ONLY

To compile

. build/envsetup.sh && lunch omni_X01BD-eng && make -j8 recoveryimage

The zenfone max pro m2 (codenamed _"X01BD"_) are high-end mid-range smartphones from asus.
